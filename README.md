# Basic-Spam-Detector
This is a simple Spam Message Detector project built using Python and basic Machine Learning. This project aims to classify messages as spam or not spam. It helps in understanding how machine learning works in real-life applications.


Overview of this project:

This is a simple Spam Message Detector built using Python.
It can classify messages as "Spam" or "Not Spam" using a basic Machine Learning algorithm.
The project uses "Naive Bayes" and "CountVectorizer" from the sklearn library using Python programming language.

Objective:-

The main goal of this project is to:

-Understand basic Machine Learning concepts
-Learn how to classify text data
-Build a simple real-world application
-Helping detect spam messages

Technologies Used:-
-Python
-scikit-learn library
-Basic machine learning algorithms: Naive Bayes


 How It Works:-

1. We give some example messages (training data)
2. Label them as spam or not spam
3. Convert text into numbers using CountVectorizer
4. Train the model using Naive Bayes
5. Test the model with new messages


How to Run the Project:-

1. Install required library

pip install scikit-learn

2. Run the Python file

 Spam_Detector.py

Example Output

Free money -> spam
Hello bro -> not spam
